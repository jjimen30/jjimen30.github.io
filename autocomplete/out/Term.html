An immutable data type Term.java that represents an autocomplete term: a string query and an associated real-valued weight. supports comparing terms by three different orders: lexicographic order by query string (the natural order); in descending order by weight (an alternate order); and lexicographic order by query string but using only the first r characters (a family of alternate orderings).
